PBI-005 Requerimiento Funcional Como usuario final, el sistema debe permitir registrar tres formas en las que el usuario considera que puede aportar al mundo, guardando esta información en el sistema para su posterior consulta o edición. Alta 1-2 semanas
  • El sistema debe mostrar tres campos de entrada para los aportes del usuario.
  • Debe existir un botón "Guardar aporte" para almacenar la información ingresada.
  • Confirmación de guardado exitoso al usuario.
